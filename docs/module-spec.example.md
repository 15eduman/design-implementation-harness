# Module Spec Example

Use this file as a placeholder for product-specific module rules.

Replace it with source-derived values from your own Figma design:

- source node ids,
- component names,
- fixed dimensions,
- spacing rules,
- image slot rules,
- variants,
- editable properties,
- locked geometry.

The module spec should not be a generic style guide. It should describe the
source-backed modules that agents must reuse or preserve.
